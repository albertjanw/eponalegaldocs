# Update Doclib Managed Navigation

Use this format to update the Managed Navigation in a document library. The filename should contain *DoclibManagedNavigation* and format is xls(x)

The following columns are required:

- Url (optional)

- ClientCode (optional)

- MatterCode (optional)

- DoclibSet
  
- ActivateFeature

If *Url* is empty, the *MatterCode* is required.

If *ActivateFeature* is *True*, the site feature Metadata Navigation and Filtering is enabled if not yet activated.
